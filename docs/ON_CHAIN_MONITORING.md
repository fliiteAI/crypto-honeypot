# On-Chain Monitoring Guide: Crypto Wallet Honeypot

This guide explains how to set up watchlists and alerts for the public addresses generated by the Crypto Wallet Honeypot.

## Overview

The final layer of detection is monitoring the public blockchain for activity on the honeypot addresses. If an attacker successfully exfiltrates a private key, they will likely import it into a wallet and query the address for funds or attempt to send tokens to it.

## 1. Export Public Addresses

Use the `honeypot-deployer` CLI to export the public addresses from your manifest into a JSON format.

```bash
honeypot-deployer export-addresses \
  --manifest ./honeypot-artifacts/manifest.json \
  --output ./watch-addresses.json
```

## 2. Set Up Watchlists

Import the exported addresses into your preferred block explorer's watchlist service.

### Ethereum and EVM Chains (Etherscan)
- **Service:** [Etherscan Watchlist](https://etherscan.io/myaddress)
- **Instructions:**
    1. Create an Etherscan account.
    2. Go to "My Profile" > "Watchlist".
    3. Add each honeypot address from `watch-addresses.json`.
    4. Enable "Email Notifications" for all incoming and outgoing transactions.

### Solana (Solscan)
- **Service:** [Solscan](https://solscan.io/)
- **Instructions:**
    1. Log in to Solscan.
    2. Add the SOL addresses to your "Account Monitor".
    3. Set up alerts for balance changes or token transfers.

### Bitcoin (Blockchain.com / Mempool.space)
- **Service:** [Mempool.space](https://mempool.space/)
- **Instructions:**
    1. Use a service that supports address notifications.
    2. Monitor for any transaction appearing in the mempool for the given addresses.

## 3. Integrating with Wazuh

For advanced users, you can script a custom integration that periodically checks these addresses using an API (e.g., Etherscan API) and sends the results back to the Wazuh Manager as a log event.

### Example Python Snippet (Etherscan API)

```python
import requests
import json

ETHERSCAN_API_KEY = "your_api_key"
ADDRESS = "0x..."

url = f"https://api.etherscan.io/api?module=account&action=txlist&address={ADDRESS}&startblock=0&endblock=99999999&page=1&offset=10&sort=asc&apikey={ETHERSCAN_API_KEY}"

response = requests.get(url)
data = response.json()

if data['result']:
    # Send this to Wazuh
    print(json.dumps({"alert": "on_chain_activity", "address": ADDRESS, "tx": data['result'][0]}))
```

## Security Best Practices

- **Never Deposit Funds:** The addresses are for detection only. Depositing real funds increases the risk of loss.
- **Rotate Manifests:** If a manifest is compromised or an address is "burned" by a detection, generate a new set of honeypots and update your watchlists.
